📊 Data Visualization Insights

This project uses Python’s Matplotlib library to visualize and analyze key business and statistical data.

📈 Sales vs Revenue

Both sales and revenue show a steady upward trend, with strong growth in the second half of the year and a stable relationship between them.

📊 Job Attrition

Highest attrition occurs in the 26–35 age group, while older employees show more stability, highlighting the need for mid-career retention strategies.

🥧 Programming Languages

Python leads with 40% usage, followed by Java and JavaScript, showing strong dominance in modern development.

📊 Distribution Analysis

Higher variance leads to wider data spread, while lower variance results in more consistent values.

📉 Height vs Weight

A clear positive correlation is observed, demonstrating how related variables move together in real-world data.
